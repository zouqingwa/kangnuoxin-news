国家人工智能应用上海中试基地（医疗领域）发布五大核心创新成果

产品技术
核心内容

3月28日，"中试为壤智创未来"国家人工智能应用上海中试基地（医疗领域）阶段性成果发布会在复旦大学附属中山医院举行。该基地由国家发展改革委、国家卫生健康委指导建设，现场发布五大核心创新成果、九款医疗智能应用。

五大核心成果包括：一体化新型算力服务平台（"国模用国芯"，性能比肩国际一流水平）、六大医疗垂直基础模型（覆盖影像、病理、中医药、科研等多元场景）、全国示范性医疗AI数据基础设施、全球领先的中文医疗大模型测试平台MedBench4.0、以及多学科世界级医疗智能应用遴选推广体系。

九款应用中，中山医院心血管病"观心"辅助诊疗智能体精准赋能心内科疑难重症诊疗；仁济医院泌尿外科智能体累计服务超50万人次；华山医院基于脑电大模型转化的侵入式脑机接口获全球首款NMPA三类证。基地与4个区、8家科研院所、28家医院、20家领军企业完成生态签约。

关注价值
1. 算力与模型生态成型：国家级中试基地提供贯通"模型研发-验证迭代-推理服务-安全治理"全链条平台，降低AI医疗企业研发门槛

2. 心血管AI走在前列：中山医院心血管"观心"智能体已在疑难重症诊疗中落地验证，心血管AI从研究走向临床
3. 政产学研用协同：28家医院+20家领军企业的生态签约规模，预示医疗AI将从单点突破迈向系统集成
4. 测试标准建立：MedBench4.0建立权威统一的医疗大模型测评标准，为AI医疗器械注册审评提供评估依据

康诺芯应关注国家级中试基地的生态合作机会，特别是心血管AI在临床验证和审批加速方面的资源支持。

Anumana获FDA批准——全球首个基于标准12导联ECG的AI肺高压检测算法

产品技术
核心内容

3月28日，心血管AI公司Anumana宣布获得美国FDA 510(k)审批，其肺动脉高压（PH）AI检测算法成为全球首个获批用于标准12导联心电图（ECG）的肺高压检测软件医疗器械（SaMD）。

该算法此前已获得FDA突破性设备认定。肺动脉高压是一种影响肺部和心脏右侧的进行性疾病，全球约1%人口受影响，但因早期症状不特异，诊断延迟常超过两年。Anumana的算法能检测标准12导联ECG中人眼难以察觉的微细异常，辅助医生判断是否需要进一步检查（如超声心动图）。临床数据亮眼：在21,066名患者的独立多中心研究中，73%敏感度和74.4%特异度；在真实世界分析中，PAH检出率超85%，CTEPH检出率78%。

关注价值
1. ECG-AI诊断新范式：从传统房颤/心衰检测扩展到肺高压检测，证明AI能从常规ECG中挖掘出全新的疾病诊断信息

2. 技术路线启发：利用超过25万例Mayo Clinic去标识化患者记录训练，展示了高质量临床数据在AI算法开发中的核心价值
3. 全球领先地位：首个标准12导联ECG肺高压算法，无需特殊设备，可广泛接入现有ECG工作流，商业化路径清晰
4. 已获报销资格：该算法及此前的LEF（左室射血分数降低）算法均已在美国获得医保报销资格，验证了AI诊断的商业化模式


康诺芯应关注ECG-AI从心脏节律分析向更多疾病领域扩展的技术趋势，评估AI算法在心电数据深层分析中的潜力。

平安集团发布"筛、管、治、康"全病程医疗AI体系——MDT诊疗方案准确率近90%

产品技术
核心内容

3月27日，2026中关村论坛"人工智能+产业"论坛上，平安集团首席技术官王晓航发表主题演讲，系统介绍平安AI在医疗领域的最新布局。

平安提出医疗AI三大技术范式演进：Scaling Law（从预训练到多智能体协同）、临床数据飞轮（千万级患者数据+每周万条医生反馈形成闭环）、真实世界对齐（RWS对齐专家鉴别思维）。核心成果：AI医生覆盖超11,300种疾病，辅助诊断准确率95.1%；在肿瘤MDT场景中，AI方案与专家组一致性超90%；"平安灵眸"构建90余种疾病筛查模型，已下沉至1,500家基层医疗机构，累计完成150万次早筛；"平安管家"构建服务200万患者的国内最大慢病管理社区。

关注价值

1. 医疗AI从外围走向诊疗内核：从信息获取/咨询深入到诊断与治疗决策，MDT一致性超90%标志着AI在严肃医疗场景的突破
2. 数据飞轮模型值得借鉴：千万级患者数据+每周万条医生反馈→强化学习持续纠偏，形成"使用驱动效果、效果吸引更多使用"的正向循环
3. 基层下沉成果显著：1,500家基层医疗机构的早筛覆盖，证明医疗AI在分级诊疗中的实际价值
4. 慢病管理社区化：200万患者慢病管理社区，AI主动干预使患者依从性提升5倍

康诺芯可借鉴平安的数据飞轮模型，思考如何将心电/血压连续监测数据转化为可持续迭代的AI诊断能力。

艾科脉医疗完成超亿元Pre-B轮融资——心脏电生理国产替代加速

资本动态
核心内容
36氪3月13日首发报道，心脏电生理领域平台型企业「艾科脉医疗」完成超亿元Pre-B轮融资，由中金资本旗下基金领投，元璟资本及知名产业投资人跟投。资金将用于脉冲电场消融（PFA）系统、三维标测系统及系列电生理耗材的研发和商业化落地。

艾科脉2020年成立，专注新一代PFA消融技术，该技术通过高压脉冲电场选择性作用于心肌细胞，避免损伤毗邻神经血管。2025年商业化首年即实现数千万元营收，PFA产品在全国近百家医院落地。核心数据：术中即刻肺静脉隔离率100%，12个月随访成功率87.7%。三维标测系统已推进注册性研究，结合AI算法实现微小电信号识别和手术流程简化。

关注价值

1. 心脏电生理赛道持续火热：超亿元融资+中金领投，表明心脏电生理国产替代获得顶级资本持续加注
2. PFA技术已成主流方向：从传统射频/冷冻消融转向脉冲电场消融，国产企业正在实现"弯道超车"
3. AI与电生理深度融合：三维标测系统利用AI处理海量手术电信号数据，实现微小病灶前兆识别和手术流程简化
4. 商业化验证：首年营收数千万元、北京集采中标、近百家医院落地，验证了国产电生理产品的市场接受度

康诺芯应关注心脏电生理领域AI应用的发展方向，以及国产医疗器械在集采环境下的商业化模式。

2026年中国智能穿戴设备市场规模预计达380亿元——医疗健康类设备年复合增长率37.2%

市场洞察
核心内容

3月26日，新浪财经发布《2026年中国智能穿戴行业发展前景预测及投资策略研究报告》。报告核心数据：2023年中国智能穿戴设备出货量1.48亿台，市场规模约1,260亿元；2025年医疗健康类穿戴设备市场规模预计达380亿元，占整体市场28.6%，较2021年提升11.2个百分点，年复合增长率37.2%。

技术趋势方面：2026年设备平均集成传感器数将从5.2增至8.7个，端侧AI推理占比超70%；华为"盘古健康"大模型支持设备端实时推理12类健康风险，延迟低于200毫秒。政策层面：国家药监局已建立AI医疗器械审评体系，17款国产设备进入创新审批通道；浙江、江苏等地已实现可穿戴设备费用50%医保报销。竞争格局：CR5达68.7%，华为25.4%、小米19.8%、苹果12.6%主导市场，专业企业如乐心医疗、硅基仿生在ECG/CGM等医疗细分赛道快速崛起。

关注价值

1. 医疗级设备增速远超消费级：37.2%年复合增长率，远超整体市场14.5%增速，医疗级赛道是明确的价值高地
2. 端侧AI推理占比将超70%：AI从云端向设备端迁移趋势明确，对芯片算力和算法轻量化提出新要求
3. 医保报销试点扩大：浙江、江苏50%报销比例，为医疗级可穿戴设备的规模化推广提供了支付路径
4. 传感器融合加速：设备从5.2增至8.7个传感器，多模态数据融合为AI算法提供更丰富的特征输入

康诺芯应把握医疗级可穿戴设备37.2%年增速的赛道红利，在端侧AI算法和传感器融合方面建立技术壁垒。
