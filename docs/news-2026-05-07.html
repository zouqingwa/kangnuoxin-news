2026年全国卫生健康体改工作会议召开——深化医改信号指向"强基层"与"数智化"

政策法规
核心内容

5月6日，2026年全国卫生健康体改工作会议在京召开。国家卫生健康委党组成员、副主任，国家中医药局党组书记、局长刘金峰出席会议并讲话。会议在全国卫生健康系统年度工作部署的关键节点召开，为下一阶段深化医药卫生体制改革定调。

从会议时间节点和与会领导级别来看，本次体改工作会议是继4月13日国务院政策例行吹风会（宣布县域医共体全覆盖）之后的又一次高层级医改会议。行业分析认为，会议释放的核心信号可能延续"强基层、促协同、推数智"三大政策主线：强基层意味着继续推进紧密型县域医共体内涵建设；促协同指向分级诊疗体系的实质性运作；推数智则与AI辅助诊断纳入医保、基层医疗机构智慧化建设等前期政策形成联动。

关注价值
本条涉及【域D-医共体/基层 + 域E-政策】方向。

核心信息：全国卫生健康体改工作会议5月6日召开，深化医改信号密集释放。

对康诺芯的意义：体改工作会议是年度医改政策走向的风向标。从4月县域医共体全覆盖到5月体改工作会议，政策节奏显示医改正在从"搭框架"阶段进入"填内容"阶段——这意味着基层医疗机构的设备采购、信息化建设、服务能力提升将从"规划期"进入"执行期"。对康诺芯而言，这是将可穿戴监测设备纳入基层医疗标准化配置清单的关键窗口。

    全国首款医疗大模型产品入选NMPA创新审查通道——胸部CT智能体诊断准确率达97.8%

    政策法规
    核心内容

    国家人工智能应用中试基地（医疗领域，简称"上海中试基地"）中试推广的智能应用——胸部CT图像辅助诊断软件，近日成功进入国家药监局创新医疗器械特别审查程序。这是全国首款进入该通道的医疗大模型产品，标志着医疗AI从"技术验证"走向"监管准入"的关键一步。

    该智能体由上海中试基地孵化，可基于单次胸部CT影像识别近百种异常病变，诊断准确率达97.8%，并自动生成媲美初年资医生水平的诊断报告。其核心突破在于将传统"1名报告医生+1名审核医生"的阅片模式转变为"AI辅助+1名医生审核"的高效模式，有望大幅提升基层医疗机构的影像诊断能力。

    该产品已在上海多家头部三甲医院（含瑞金医院、中山医院等）完成临床验证。上观新闻、东方网、新浪等多家媒体报道了这一突破。科学网（中国科学报社）指出，此举为国内医疗大模型产品获批医疗器械注册证打开了通道。

    关注价值
    本条涉及【域B-AI医疗 + 域E-政策/监管】方向。

    核心信息：首款医疗大模型产品进入NMPA创新审查通道，诊断准确率97.8%，在三甲医院完成临床验证。

    对康诺芯的意义：医疗大模型产品首次进入NMPA审查程序是一个标志性事件——它意味着AI辅助诊断的监管框架正在从"无规可依"走向"有章可循"。对康诺芯而言，这一进展为AI心电分析算法的注册提供了可参照的路径。特别值得注意的是，97.8%的诊断准确率和"AI辅助+医生审核"模式，为AI诊断产品的临床定位提供了行业基准——不是替代医生，而是提升效率和均质化水平。

    县域医共体信息化政策下：普瑞森健康一体机的场景价值与系统定位

    政策法规
    核心内容

    随着县域医共体信息化建设深入推进，健康一体机作为基层医疗数据采集的关键终端设备，其场景价值日益凸显。普瑞森健康一体机集成血压、血糖、心电、体温等多项检测功能，通过一次操作实现多项数据自动采集与上传，减少人工录入误差。

    该设备采用场景化设计——以基层医生实际工作流程为导向，界面与操作简化，降低使用门槛；支持与医共体信息平台无缝对接，实现检测数据实时上传、异常预警和远程会诊，为慢病管理、健康档案建立提供技术支撑。目前已在多个县域医共体试点中应用。

    行业对比分析指出，2026年健康一体机市场已形成"多参数集成+数据互联+场景适配"的竞争格局，尚泰、普瑞森、乐普等厂商各有侧重。设备选型的核心标准已从"参数多不多"转向"数据通不通"——能否与医共体信息平台对接、是否支持医保数据上报、是否具备远程诊断接口，正在成为采购决策的关键考量。

    关注价值
    本条涉及【域D-医共体/基层 + 域A-可穿戴监测】方向。

    核心信息：健康一体机正成为县域医共体信息化的数据采集终端，选型标准从"参数竞争"转向"互联互通能力"。

    对康诺芯的意义：健康一体机是基层医疗数据采集的"入口级"设备。康诺芯虽不直接生产一体机，但其动态心电监测、腕式血氧等可穿戴设备可以作为一体机的"扩展外设"——一体机做院内快速筛查，可穿戴设备做院外连续监测。这种"院内+院外"组合拳正是医共体全周期健康管理的数据基础设施。建议关注各地医共体的设备采购标准，确保康诺芯产品与主流一体机品牌的数据接口兼容。

    "十五五"时期人均预期寿命提到80岁——健康规划对可穿戴和慢病管理的顶层定调

    政策法规
    核心内容

    "十五五"时期，中国人均预期寿命目标提升至80岁。这一规划目标的设定，标志着中国健康战略的核心命题从"延长寿命"升级为"提升有质量的寿命"——不仅要活得长，还要活得好。

    报道指出，实现80岁人均预期寿命目标的关键路径包括：控制高血压、糖尿病等慢性病，遵医嘱规范管理，将指标控制在理想范围；可穿戴设备、健康APP、远程医疗等数字化健康工具将更深入日常生活；学会利用这些工具进行持续监测和主动管理，是"活出有质量的寿命"的核心能力。

    这一规划目标与国务院此前印发的《关于推进服务业扩能提质的意见》（将健康管理提升至战略高度）、国家卫健委基层医疗质量改善三年行动等政策形成呼应，共同构成"十四五收官→十五五开启"阶段的健康政策框架。

    关注价值
    本条涉及【域C-慢病管理 + 域A-可穿戴监测 + 域E-政策规划】方向。

    核心信息："十五五"人均预期寿命目标80岁，可穿戴设备和慢病管理被明确纳入实现路径。

    对康诺芯的意义：国家五年规划中明确提及可穿戴设备和慢病管理的角色，是最高层级的政策背书。"人均预期寿命80岁"不是一个抽象数字——它意味着对慢病管理覆盖率、规范化率和控制率的系统性考核压力，进而转化为对可穿戴监测设备的刚性需求。对康诺芯而言，应关注两点：一是"十五五"规划中可能出现的慢病管理量化指标（如高血压控制率、糖尿病管理率），这些指标直接对应设备需求；二是"活得有质量"的表述暗示健康管理的价值评估标准正在从"治病"转向"防病+管病"，这正是可穿戴设备的核心价值区间。

    智诊科技完成6500万元天使轮融资——DoctorBench冠军获资本认可，医疗AI Agent赛道升温

    资本动态
    核心内容

    智诊科技近日完成6500万元天使轮融资，投资方为无锡元启股权投资合伙企业（有限合伙），后者聚焦科技创新与产业生态投资，重点关注人工智能、数字经济及医疗健康等领域。


    智诊科技的核心产品WiseDiag-v2曾在德适生物4月30日发布的DoctorBench医疗大模型评测平台中位列全球第一。DoctorBench以"临床思维考核"替代传统"知识问答"评测逻辑，被认为是医疗AI从"模型竞赛"进入"临床能力验证"阶段的标志性事件。智诊科技在此评测中夺冠，展示了其在临床推理能力上的领先优势。

    本轮融资将用于继续提升模型能力，并将技术积累转化为面向医疗健康产业的赋能能力和面向个人用户的健康管理服务。公司定位为"医疗AI agent"，区别于单一的影像AI或文本AI，其产品方向是构建具有临床推理能力的综合型AI辅助系统。

    关注价值
    本条涉及【域B-AI医疗 + 域E-资本动态】方向。

    核心信息：DoctorBench冠军智诊科技获6500万元天使轮融资，医疗AI Agent方向获资本认可。

    对康诺芯的意义：智诊科技融资有三层信号：第一，DoctorBench评测成绩正在成为医疗AI公司的"资本市场通行证"，说明"临床思维考核"这一评测标准已开始影响投资决策——康诺芯的AI心电分析能力也应关注类DoctorBench的评测标准；第二，"医疗AI agent"的定位比单一影像AI更具想象空间，与康诺芯"AI辅诊智能体"的产品方向有交集但不同层级（智诊做通用推理，康诺芯做垂直心电）；第三，6500万天使轮在当前融资环境下是不错的成绩，且投资方强调"技术与产业结合"，说明医疗AI赛道仍有资本热度。
